‚ùì Why Manual Activation?

GitHub Security Restrictions

GitHub prevents GitHub Apps from creating or modifying workflow files as a security measure. This prevents malicious apps from injecting code into CI/CD pipelines. The GitHub App token used by this system has read-only access and cannot push workflow files, even with admin permissions on the repository.
