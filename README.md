# Universal Build System

Build any app from any device with simple commands.

## Quick Start

1. **Preview locally:**
   ```bash
   ./preview
   ```

2. **Deploy to Cloudflare:**
   ```bash
   ./deploy my-app-name
   ```

## Works From Anywhere

- Claude.ai (web browser)
- Claude Desktop
- Any computer with Git installed

See `HOW_TO_USE.md` for complete documentation.
